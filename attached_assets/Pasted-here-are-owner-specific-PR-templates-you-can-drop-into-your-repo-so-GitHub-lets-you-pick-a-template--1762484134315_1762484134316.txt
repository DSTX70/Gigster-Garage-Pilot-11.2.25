here are owner-specific PR templates you can drop into your repo so GitHub lets you pick a template per PR from a dropdown.

Where to put them

Create this folder (exact path):

.github/PULL_REQUEST_TEMPLATE/


Add one Markdown file per owner (filenames below). GitHub will show these as selectable templates when opening a PR. You can keep your existing global template too (as .github/pull_request_template.md)—GitHub will let you choose.
