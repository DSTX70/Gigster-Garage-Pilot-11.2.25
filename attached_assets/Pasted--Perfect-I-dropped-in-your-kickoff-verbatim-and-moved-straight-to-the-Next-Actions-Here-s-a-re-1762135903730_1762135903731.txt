### Run steps (Replit or local)

1. **Start the Hub (port 8000)**

```bash
pip install -r hub/requirements.txt
uvicorn hub.main:app --host 0.0.0.0 --port 8000 --reload
```

2. **Run Switchboard (Node 18+)**

```bash
npm install
npm run switchboard
```

This writes **public/visibility_flags.json** based on KPI thresholds or `flip_schedule.json`.

3. **Point your Vite app**

```
VITE_HUB_BASE_URL=http://localhost:8000
VITE_USE_MOCKS=false
VITE_MOCK_FALLBACK=true
VITE_DEMO_RUN_IDS=demo-1,demo-2,demo-3
VITE_ADMIN_WRITE=true   # enables UI ‚ÄúPromote‚Äù POSTs to /admin/flags/{agent}
```
